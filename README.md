# BananaBet — Decentralized P2P Betting Platform

BananaBet — навчальний проєкт, розроблений у межах лабораторної роботи з дисципліни
«Технологія блокчейн та машинне навчання».

Проєкт реалізує децентралізовану P2P betting-платформу без букмекерської маржі
з використанням машинного навчання та блокчейн-технологій.

## Архітектура системи

- **ML-сервіс (FastAPI)** — прогнозування ймовірностей та розрахунок fair odds
- **Backend-оракул (.NET)** — інтеграція ML, БД та блокчейну
- **Смарт-контракти (Solidity)** — матчинг ставок, зберігання балансів, виплати
- **Frontend (React + Vite)** — користувацький Web3-інтерфейс

## Структура репозиторію

- `ml-service/` — ML-модуль
- `backend-oracle/` — backend-оракул
- `smart-contracts/` — смарт-контракти BananaUSD та BananaBet
- `frontend/` — frontend-клієнт

## Тестування

Система протестована вручну:
- окремо ML-модуль (Swagger)
- окремо смарт-контракти (Remix IDE)
- повний end-to-end цикл (ML → Backend → Blockchain → Frontend)

## Примітка

Проєкт створено в навчальних цілях. Не призначений для використання з реальними коштами.
